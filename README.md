
# CarePLUS

This project is a full responsive hospital website built using HTML, SASS, CSS, and JavaScript. It aims to provide a user-friendly and informative interface for patients and visitors. 



  
## Technologies Used

- HTML

- SASS (SCSS) 

- CSS

- JavaScript

## Usage
- Explore the different sections of the website, including the homepage, services, doctors, gallery, and contact page.
- Book an appointment by filling out the online form in the "Appointments" section.
- Use the contact form to send inquiries or feedback to the hospital.
